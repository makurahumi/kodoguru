# Kodogru
## 概要
お気に入りの飲食店やお弁当販売店などをレビュー形式で投稿、共有、閲覧できるアプリです。
お店の外観と住所など詳細情報とお店のメニューに関する情報をそれぞれ投稿する事ができます。
また、マイページでは自分の投稿したお店の情報を確認できます。
